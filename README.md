# 🐋 Pump.Fun Whale Detector

یک وب‌سایت تخصصی و دقیق برای شناسایی نهنگ‌ها و تحلیل احتمال Pump/Dump در توکن‌های Pump.fun

## ✨ ویژگی‌ها

- 🐋 **شناسایی نهنگ‌ها** - تشخیص خریدهای بزرگ قبل از Pump
- 📊 **تحلیل حجم معاملات** - بررسی حجم و نقدینگی Real-time
- 💧 **تحلیل انباشت** - ردیابی تجمع سرمایه
- ⚠️ **هشدار خطر** - اطلاع‌رسانی فوری از Pump/Dump
- 📈 **نمودارهای تحلیلی** - داشبورد جامع و دقیق
- 🎯 **الگوریتم پیشرفته** - تحلیل عمیق داده‌ها

## 🚀 تکنولوژی

- **Frontend:** React + TypeScript
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **API:** Pump.fun API + On-chain Data
- **Real-time:** WebSocket

## 📁 ساختار پروژه

```
pump-fun-whale-detector/
├── frontend/          # رابط کاربری React
├── backend/           # سرور Node.js
├── data/              # تحلیل داده‌ها
├── config/            # تنظیمات
└── docs/              # مستندات
```

## 🔧 نصب و اجرا

```bash
# نصب وابستگی‌ها
npm install

# اجرای سرور
npm run dev

# ساخت برای Production
npm run build
```
